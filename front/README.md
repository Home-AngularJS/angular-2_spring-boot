## Installation
* Need
  * `node -v` **v10.16.0**
  * `npm -v` **6.9.0**
  * `ng version` **Angular CLI: 8.1.1**
* `npm install` to install Node packages
* `npm install -g @angular/cli` to install [Angular CLI](https://github.com/angular/angular-cli) ( globally - если уже установлен тогда не обязательно) 
* Default port **4200**
